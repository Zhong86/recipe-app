# Recipe App

A web application for sharing and discovering recipes, built with Laravel.

## Features
- Browse recipes
- Create an account and post your own recipes
- Save favorites
- Search and filter by category

## Tech Stack
- Laravel (PHP)
- MySQL
- Bootstrap CSS
- AWS S3 for image storage
